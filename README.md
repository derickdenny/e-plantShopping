# e-plantShopping 🌱

## Paradise Nursery - React + Redux Project

This project is a plant shopping web application built using React and Redux. Users can browse different categories of plants, add them to a shopping cart, and manage their cart items.

## Features

- Display plants grouped by categories
- Add to cart functionality
- Dynamic cart item count
- Increase / decrease quantity
- Remove items from cart
- Total cart cost calculation
- Responsive UI

## Technologies Used

- React.js
- Redux Toolkit
- CSS

## Project Structure

- ProductList.jsx → Displays plants
- CartItem.jsx → Shopping cart page
- CartSlice.jsx → Redux logic
- App.jsx → Landing page
- App.css → Styling
